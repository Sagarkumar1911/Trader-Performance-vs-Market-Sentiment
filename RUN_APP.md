# Running the Streamlit App

## Quick Start

To launch the sentiment-based trader performance app:

```bash
streamlit run app.py
```

The app will open at `http://localhost:8501/` by default.

## Features

### Tab 1: Sentiment Analysis
- **Sentiment Dropdown**: Select "Fear" or "Greed" to filter data
- **Key Metrics**: See aggregated performance for the selected sentiment
  - Average Daily PnL
  - Average Win Rate
  - Average Trade Count
  - Average Leverage
  
- **Top-Performing Segments**: Displays a table and chart showing which trader segments (Whale, Degen, Grinder, Retail) perform best
  - Sorted by Average PnL (highest first)
  - Shows median PnL, standard deviation of returns
  - Win rate comparison
  - Trade frequency analysis
  
- **Dynamic Charts**:
  - Segment Rankings (bar chart of average PnL)
  - Win Rate Comparison (bar chart with 50% baseline)
  - Trade Frequency by Segment
  - Detailed breakdown table with min/max values

### Tab 2: Account Search
- **Account Selector**: Dropdown to search for any specific trading account
- **Account Summary**: 5-metric overview
  - Total PnL (cumulative)
  - Average Daily PnL
  - Overall Win Rate
  - Total Trade Count
  - Total Trading Days
  
- **Account Profile**: Detailed information
  - Trader Segment (Whale/Degen/Grinder/Retail)
  - Cluster Assignment (from K-means)
  - Average Trade Size
  - Average Leverage
  - Long Ratio (buy bias)
  - PnL Volatility (standard deviation)
  
- **Performance Breakdown**: Shows how each account performs by sentiment
  
- **Daily History Table**: Search and view complete trading history
  - Sortable by any column
  - Shows all key metrics per day
  - Color-coded for USD values and percentages
  
- **Cumulative PnL Chart**: Visualize profit/loss progression over time
  
- **Sentiment Impact**: Two charts showing
  - Average PnL by sentiment
  - Win rate by sentiment

## Data Used

The app loads from these files generated by `analysis_clean.py`:
- `outputs/daily_trader_summary.csv` - Daily performance data (2,340 records)
- Required columns: Account, date, sentiment_binary, total_pnl, win_rate, trade_count, leverage_proxy, etc.

## Customization

Edit app.py to:
- Change color scheme (BINARY_PALETTE dictionary)
- Modify layout (use `st.set_page_config()`)
- Add more sentiment options (currently Fear/Greed)
- Add filtering options (date range, leverage range, etc.)

## Tips for Recruiters

1. **Find Top Performers**: Use Sentiment Analysis tab to see which segments outperform
2. **Deep Dive History**: Use Account Search to investigate specific trader performance
3. **Sentiment Sensitivity**: Check if traders perform differently under Fear vs Greed
4. **Consistency**: Look for accounts with stable performance across both sentiments
5. **Risk Management**: Use Leverage and Win Rate metrics to assess risk profiles

## If App Won't Load

```bash
# Clear Streamlit cache
rm ~/.streamlit/cache_*

# Reinstall streamlit
pip install --upgrade streamlit

# Run with debug info
streamlit run app.py --logger.level=debug
```
